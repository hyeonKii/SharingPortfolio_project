# 포트폴리오 서비스
---

|역할|이름|
|---|---|
|백엔드|김현정|
|백엔드|김병용|
|백엔드|김종훈|
|프론트엔드|김현기|
|프론트엔드|박은령|
|프론트엔드||


5개 MVP로 구성

- User (회원가입, 로그인 등 사용자 관련)
- Award (포트폴리오 중 상장 이력 관련)
- Certificate (포트폴리오 중 자격증 관련)
- Project (포트폴리오 중 프로젝트 관련)
- Education (포트폴리오 중 교육, 학교 관련)

## 주요 사용 기술

1. 프론트엔드

- React (create-react-app으로 구현)
- React Bootstrap
- axios

2. 백엔드

- Express (nodemon, babel-node로 실행)
- Mongodb, Mongoose

## 설치 방법

1. 프론트 엔드 서버 실행

```bash
cd front
npm i
npm start
```

2. 백엔드 서버 실행

```bash
cd back
npm i
npm start
```
